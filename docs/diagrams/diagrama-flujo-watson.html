Flujo del Chatbot Watson Assistant

Modo Online (IBM Cloud) • Modo Offline (palabras clave) • Resiliencia 24/7

sequenceDiagram
    autonumber
    actor Cliente
    participant Frontend Widget
    participant WatsonController
    participant WatsonService
    participant IBM Watson Cloud

    Cliente->>Frontend Widget: Escribe consulta: "¿Qué autos hay disponibles?"
    Frontend Widget->>WatsonController: POST /api/watson/message
    WatsonController->>WatsonService: procesarMensaje(mensaje, sessionId)
    WatsonService->>WatsonService: ¿Credenciales Watson configuradas?

    alt Modo ONLINE
        WatsonService->>IBM Watson Cloud: POST /v2/assistants/{id}/sessions/message
        IBM Watson Cloud-->>WatsonService: Respuesta del asistente entrenado
        WatsonService-->>WatsonController: respuestaOnline()
        WatsonController-->>Frontend Widget: 200 OK { esOffline: false }
        Frontend Widget-->>Cliente: 🤖 Respuesta de Watson
    else Modo OFFLINE
        WatsonService->>WatsonService: Palabras clave: disponibilidad, precios...
        alt disponibilidad, autos
            WatsonService->>WatsonService: DISPONIBLES
        else precio, cuesta
            WatsonService->>WatsonService: PRECIOS
        else reservar, alquilar
            WatsonService->>WatsonService: RESERVAR
        else hola, saludo
            WatsonService->>WatsonService: SALUDO
        else otra consulta
            WatsonService->>WatsonService: Respuesta por defecto
        end
        WatsonService-->>WatsonController: respuestaOffline()
        WatsonController-->>Frontend Widget: 200 OK { esOffline: true }
        Frontend Widget-->>Cliente: 🤖 Respuesta predefinida
    end
    Note over WatsonService,IBM Watson Cloud: 🛡️ Modo offline: chatbot responde aunque IBM Cloud no esté disponible.
